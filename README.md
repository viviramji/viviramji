## Architecture Principles I Follow

```
✔  No SOQL or DML inside loops — always bulkified
✔  WITH USER_MODE / Security.stripInaccessible() on every query
✔  Trigger Action Framework — no logic in triggers directly
✔  LDS over imperative Apex where possible
✔  @isTest with meaningful assertions — never seeAllData=true
✔  Declarative first (Flow/Configuration), Apex only for complex logic
```
